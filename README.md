# FireOS

OS made as an academic project. This is a first-try attempt to create a functional os from the scratch.

Authors: Vadzim Shyshko, Damian Duy

# Requirements
- qemu
- gcc cross-compiler

# Building and running
To build execute:
```
make
```
To run execute:
```
make run
```
To clear leftover files execute:
```
make clear
```
